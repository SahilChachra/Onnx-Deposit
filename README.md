## :innocent: Motivation
A single repo with code & some theory to quickly refer when working with Onnx.

        </ul>

        <ul>

        <li>UNet - Onnx inference script</li>
    </ul>
</ol>

<ol>

    <li>Checkout notes on the top of the scripts before running the codes</li>
</ol>
